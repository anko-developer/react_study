## Clean Coding

- **아래 3가지는 반드시 지켜야 할 것들 이라고 생각한다.**

  ```
  1. DRY 하게, (Don`t Repeat Yourself),
  계속 반복되어지는 것을 피하자.
  2. KISS 하게, (Keep It Simple, Stupid),
  코드는 심플하고 멍청하게.
  3. YAGNI 하게, (You Ain`t Gonna Need It),
  깨끗하게, 변경이 쉽게, 유지보수 용이하게 (O)
  필요하지 않는 기능, 사용하지 않는 기능, 지나치게 미래지향적 (X)
  ```
